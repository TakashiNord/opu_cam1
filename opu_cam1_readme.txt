cam1

Purpose:
  Программа для изменения параметров операции.
  Меняет тип движения (линейные-круговые-сплайны).

Requirements for this package:
  UG V18/NX1/NX2/....
  Windows NT/2000/XP
  UG/Gateway
  UG/Manufacturing
  UG/Open API Execute

Restrictions:
  not

Author:
  ЧЕ

Quick installation:
   Move 'cam1.dll' to one of the following directories:
     %UGII_USER_DIR%\application 
     %UGII_SITE_DIR%\application 
     %UGII_VENDOR_DIR%\application 
  - for calling 'cam1.dll' via File->Execute UG/Open->User Function or User Tool:
      Move 'cam1.dll' to one of the following directories:
        %UGII_USER_DIR%\application 
        %UGII_SITE_DIR%\application 
        %UGII_VENDOR_DIR%\application 

  The required enironment-variable (i.e. UGII_USER_DIR or UGII_SITE_DIR or
  UGII_VENDOR_DIR) must be set properly on your system.

Most common problem:
